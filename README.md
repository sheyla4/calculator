# Calculator Project

A basic Python Calculator application that performs fundamental operations.

## Features 
*✶ Addition ('add')
*✶ Subtraction ('sub')
*✶ Multiplication ('mul')
*✶ Division ('div')
*✶ Power ('pow')
*✶ Square Root ('sqrt')

All operations are performed solely between two numbers.

## Quality Control
This project includes **GitHub Actions** to automatically check Python code complies with proper styling and formatting.
